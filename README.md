## ğŸ“š Curriculum Overview

### Kindergarten (23 Lessons)
- **Counting & Cardinality** - Count to 20, write numbers, compare quantities
- **Operations & Algebraic Thinking** - Understand addition/subtraction, make 10
- **Number & Operations in Base Ten** - Teen numbers (11-19)
- **Measurement & Data** - Describe and compare objects, classify and count
- **Geometry** - Identify and describe 2D/3D shapes

### Grade 1 (24 Lessons)
- **Operations & Algebraic Thinking** - Add/subtract within 20, word problems, properties
- **Number & Operations in Base Ten** - Place value, count to 120, add within 100
- **Measurement & Data** - Order by length, tell time, organize data
- **Geometry** - Distinguish attributes, partition shapes

### Grade 2 (25 Lessons)
- **Operations & Algebraic Thinking** - Add/subtract within 100, even/odd, arrays
- **Number & Operations in Base Ten** - Place value to 1000, fluency within 100
- **Measurement & Data** - Standard units, time, money, line plots, bar graphs
- **Geometry** - Recognize shapes, partition rectangles

### Grade 3 (26 Lessons)
- **Operations & Algebraic Thinking** - Multiplication, division, two-step problems, patterns
- **Number & Operations - Fractions** - Unit fractions, number lines, equivalent fractions
- **Number & Operations in Base Ten** - Round numbers, multiply/divide by 10
- **Measurement & Data** - Time, mass, volume, scaled graphs, area, perimeter
- **Geometry** - Understand shape categories, partition shapes

### Grade 4 (27 Lessons)
- **Operations & Algebraic Thinking** - Multiplicative comparisons, multi-step problems, factors
- **Number & Operations in Base Ten** - Place value to 1,000,000, multi-digit operations
- **Number & Operations - Fractions** - Equivalent fractions, add/subtract, decimals
- **Measurement & Data** - Convert units, area/perimeter, line plots, angles
- **Geometry** - Draw lines/angles, classify shapes, symmetry, measure angles

### Grade 5 (28 Lessons)
- **Operations & Algebraic Thinking** - Numerical expressions, order of operations, patterns
- **Number & Operations in Base Ten** - Place value patterns, powers of 10, decimal operations
- **Number & Operations - Fractions** - Add/subtract unlike denominators, multiply/divide fractions
- **Measurement & Data** - Convert units, line plots, volume
- **Geometry** - Coordinate plane, classify 2D figures hierarchically

### Grade 6 (29 Lessons)
- **Ratios & Proportional Relationships** - Understand ratios, unit rates, percent
- **The Number System** - Divide fractions, integers, absolute value, rational numbers
- **Expressions & Equations** - Write/evaluate expressions, solve equations, inequalities
- **Geometry** - Area of polygons, volume, coordinate plane, nets, surface area
- **Statistics & Probability** - Statistical questions, data distributions, mean/median/mode

## ğŸš€ Deployment

This site is deployed using **GitHub Pages**.

### Setup Instructions

1. **Clone or download this repository**
   ```bash
   git clone https://github.com/starhyeon/StarInBorderland.git
   ```

2. **Ensure all files are present:**
   - All `.html` files
   - `styles.css`
   - `script.js`

3. **Deploy to GitHub Pages:**
   - Go to repository Settings
   - Navigate to "Pages" section
   - Select "Deploy from branch"
   - Choose `main` branch, `/ (root)` folder
   - Save and wait 1-2 minutes

4. **Access your site:**
   - `https://yourusername.github.io/StarInBorderland/`

### Local Development

To test locally:

1. Open `index.html` in a web browser
2. Or use a local server:
   ```bash
   python -m http.server 8000
   # Then visit http://localhost:8000
   ```
